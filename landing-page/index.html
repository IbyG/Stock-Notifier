How It Works

Simple, automated workflow from configuration to notification

1

Configure

Set up your funds in funds_config.yml and configure notification preferences in the .env file.

2

Scrape

The scraper automatically visits Vanguard's website, extracts price data, and processes historical information for each configured fund.

3

Notify

Receive instant notifications with price updates, changes, and fund performance delivered to your preferred platform.

Quick Start Guide

Get up and running in minutes

1

Clone the Repository

git clone https://github.com/IbyG/Stock-Notifier.git cd Stock-Notifier
2

Configure Environment

cp .env.example .env nano .env # Add your ntfy server details
3

Configure Funds

nano funds_config.yml # Add your Vanguard funds
4

Build and Run

docker build -t stock-notifier . docker run --rm stock-notifier
